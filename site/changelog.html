Changelog

Sidetation 的每一次更新都记在这里。

v0.4.02026-07-04

官网、快照缩略图与文字编辑

  • 工具介绍官网上线（就是你现在看到的这个）
  • 修改列表每条记录新增「优化前 → 优化后」快照缩略图，原生 SVG foreignObject 实现
  • 双击纯文本元素进入行内文字编辑，改动进撤销栈与导出记录
  • 快捷键收敛为工具栏 ⌘ 按钮，hover 展示完整速查面板
  • 属性面板改为视口垂直居中展示
  • 撤销栈支持批量动作（为多选批量编辑铺路）
v0.3.02026-07-03

Figma 式属性面板

  • 悬浮属性面板：Auto layout（flex 方向 / 对齐九宫格 / gap）、Fill、Stroke、圆角、透明度
  • Padding 支持双值 / 四边独立两种模式，自动识别不对称 padding
  • 面板改动与拖拽缩放同源：进撤销栈、进修改记录、进双格式导出
v0.2.02026-07-03

Figma 基础快捷键与任意网站注入

  • 全局 undo / redo 栈（⌘Z / ⌘⇧Z），连续微调自动合并为一步
  • Delete 删除元素、Enter / Shift+Enter 下钻上钻、Tab 切换兄弟元素
  • Shift 等比缩放扩展到边手柄
  • 拖回原位的零位移记录自动过滤
  • 新增 console-paste.js 与 bookmarklet，可注入任意网站测试
  • 兼容 Trusted Types 严格站点（全量移除 innerHTML）
v0.1.02026-07-03

MVP 首发

  • 点选元素：hover 高亮、点击选中、子元素下钻、Alt+点击选父级
  • 拖拽移动：transform 位移 + Figma 式对齐吸附参考线
  • 8 手柄缩放，content-box 自动补偿
  • 方向键微调、编辑历史面板、单条撤销
  • 双格式导出：结构化 Markdown（给 AI agent）与可直接粘贴的 CSS
  • 稳定选择器生成，过滤 CSS-in-JS 哈希类名
